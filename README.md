\# 🚀 Plataforma de Cursos Online - Dashboard Premium



Um painel administrativo moderno, responsivo e focado na experiência do usuário para gestão completa de uma plataforma de ensino a distância (EAD). Desenvolvido com uma interface Premium em \*\*Dark Mode\*\*.



\## 💻 Sobre o Projeto



Este sistema foi construído para centralizar as operações de uma escola online. A aplicação é do tipo \*Single Page Application\* (SPA) simulada no frontend, gerenciando os estados e persistência de dados localmente através do `localStorage` do navegador, garantindo alta performance sem a necessidade imediata de um backend.



\## ✨ Funcionalidades



\* \*\*👥 Gestão de Usuários:\*\* Cadastro de alunos, instrutores e administradores, com controle de matrículas ativo.

\* \*\*🎓 Área Acadêmica:\*\* Criação de categorias e estruturação de novos cursos (iniciante, intermediário, avançado).

\* \*\*📚 Conteúdo e Trilhas:\*\* Upload de aulas (vídeo/texto), organização por módulos e agrupamento de cursos em trilhas de formação (ex: Formação Full-Stack).

\* \*\*💳 Módulo Financeiro:\*\* Controle de planos de assinatura, visão geral de assinaturas ativas e histórico de pagamentos (transações via Pix e Cartão de Crédito).

\* \*\*📜 Certificados:\*\* Validação de conclusão de curso e emissão de certificados com código de autenticação único.



\## 🛠️ Tecnologias Utilizadas



\* \*\*HTML5 \& CSS3:\*\* Semântica avançada e estilização customizada.

\* \*\*JavaScript (Vanilla):\*\* Lógica de negócios, manipulação do DOM e persistência de dados.

\* \*\*Bootstrap 5:\*\* Sistema de grids estrutural e componentes responsivos (Modais, Tabelas, Alertas).

\* \*\*LocalStorage API:\*\* Simulação de banco de dados relacional (JSON) no lado do cliente.

\* \*\*UI/UX:\*\* Paleta de cores \*Dark/Indigo\*, com foco em contraste, legibilidade e efeitos de hover imersivos.
